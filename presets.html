Manual override

Set the humidity you want, then let Essentia decide when the room can help.

Manual mode is for careful users: growers, cooks, bulk buyers, and collectors who already know the range they want. Essentia does not force the jar toward unsafe conditions; it only manages exchange when the outside environment can move the contents toward the target.

Example: pantry protection

Set sugar to a low target. If the kitchen gets humid during cooking, Essentia stays closed instead of letting clump-forming moisture into the jar.

Example: herbs and flower

Set a moderate target for aroma and feel. If room air would over-humidify the jar, Essentia waits for better conditions.
