Country shipping rate test.

Run Easyship rate checks by country code using Essentia's pre-production box profile. This page is not linked publicly and the endpoint requires a test token.

25 Maximum country checks per API batch to avoid timeouts and surprise Easyship usage.

Test setup

Run a batch.

A fail can mean no carrier lane, lithium battery restriction, invalid sample address, DDP not supported, or missing Easyship configuration. Confirm failures manually before blocking a country.

Package profile

One unit tests as a 6 x 6 x 4 in box at 1.1 lb, expressed to Easyship as 15.2 x 15.2 x 10.2 cm and 0.50 kg.

Required Cloudflare env vars

Set `EASYSHIP_API_TOKEN`, all `EASYSHIP_ORIGIN_*` fields, and `SHIPPING_TEST_TOKEN`. The Stripe key is not required for this diagnostic.
